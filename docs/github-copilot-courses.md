## 課程清單

### 1. 掌握 AI 驅動的終端機：GitHub Copilot CLI 深度實戰

**課程連結**: https://learn.duotify.com/courses/copilot-cli

**課程資訊**:
- 影片數量: 1 部
- 總時長: 03:43:44

**課程說明**:

從精通 Claude Code、Codex CLI、Gemini CLI 三家 AI Coding 工具之後，使用 AI 開發程式從來沒有斷炊過，哪一家中斷服務時，可以立刻切換到另一套工具。最近又多出了一個工具選項：GitHub 的 Copilot CLI 工具！

**為什麼 Copilot CLI 值得學習**:

- **GitHub 生態系的優勢**: GitHub Copilot CLI 直接整合在 GitHub 生態系中，與最新的 GitHub Copilot coding agent 無縫整合，雲地合一的開發可以替開發人員帶來更大的效能提升。

- **完全免費使用**: 對於已經訂閱 GitHub Copilot 的用戶而言，Copilot CLI 是完全免費的額外功能，不需要額外付費。

- **終端機友善**: Copilot CLI 專為終端機環境設計，可以直接在命令列中使用，非常適合習慣使用終端機的開發者。

**適合對象**: 已經使用 GitHub Copilot 的開發者、喜歡在終端機環境工作的開發者

---

### 2. GitHub Copilot 進階開發實戰

**課程連結**: https://learn.duotify.com/courses/gh-copilot-pro

**課程資訊**:
- 影片數量: 2 部
- 總時長: 07:02:30

**課程說明**:

這是 GitHub Copilot 課程的進階版本，適合已經有基礎 GitHub Copilot 使用經驗的開發者。課程內容涵蓋了 GitHub Copilot 的進階功能與實戰技巧。

**課程特色**:

通過實際案例展示如何使用 GitHub Copilot 快速開發複雜功能。例如：使用 GitHub Copilot 輔助開發網頁爬蟲程式，只需要短短五分鐘就能完成。

自《GitHub Copilot 協作開發實戰》課程推出一年以來，GitHub Copilot 持續進化，除了 LLM 基礎模型的更新，還加入了許多新功能。本課程會介紹這些最新的進階功能。

**適合對象**: 已經有 GitHub Copilot 基礎使用經驗的開發者、想要深入學習進階功能的開發者

---

### 3. GitHub Copilot 協作開發實戰

**課程連結**: https://learn.duotify.com/courses/gh-copilot

**課程資訊**:
- 影片數量: 1 部
- 總時長: 02:37:52

**課程說明**:

這是 GitHub Copilot 的入門與基礎課程，適合剛開始接觸 GitHub Copilot 的開發者。

**課程重點**:

許多開發者雖然使用 GitHub Copilot，但往往只使用到非常基礎的功能，沒有真正掌握 GitHub Copilot 的精髓之處。本課程將教你：

- 如何正確使用 GitHub Copilot 的各項功能
- 如何下達有效的 Prompt 來完成各種開發任務
- 如何充分發揮 GitHub Copilot 的協作能力

**課程目標**:

讓學員能夠從基礎使用者進階到熟練使用 GitHub Copilot，真正提升開發效率。

**適合對象**: GitHub Copilot 初學者、想要系統化學習 GitHub Copilot 的開發者

---

### 4. GitHub Copilot 代理人模式實戰

**課程連結**: https://learn.duotify.com/courses/gh-copilot-agent

**課程資訊**:
- 影片數量: 3 部
- 總時長: 06:58:58

**課程說明**:

GitHub Copilot 的 Agent Mode (代理人模式) 是一個革命性的功能，這是一個全自動的 AI Agent，可以幫助開發者解決各種複雜的開發問題。

**Agent Mode 的特色**:

- **全自動開發**: Agent Mode 可以自動分析需求、規劃解決方案、撰寫程式碼
- **複雜任務處理**: 能夠處理比一般 AI 輔助工具更複雜的開發任務
- **改變開發方式**: 這個功能將徹底改變傳統的開發方式

**課程內容**:

深入探討 Agent Mode 的使用方式、最佳實踐，以及如何將其整合到日常開發流程中。

**適合對象**: 想要了解 AI Agent 開發模式的開發者、追求最新技術的開發者

## 學習路徑建議

### 初學者路徑
1. 先學習「GitHub Copilot 協作開發實戰」(gh-copilot) 打好基礎
2. 進階到「GitHub Copilot 進階開發實戰」(gh-copilot-pro) 學習進階功能
3. 探索「GitHub Copilot 代理人模式實戰」(gh-copilot-agent) 了解最新的 AI Agent 功能

### 進階開發者路徑
1. 如果已經有 GitHub Copilot 使用經驗，可以直接學習「GitHub Copilot 進階開發實戰」
2. 對終端機開發有興趣的，可以學習「掌握 AI 驅動的終端機：GitHub Copilot CLI 深度實戰」
3. 想要掌握最新技術的，推薦「GitHub Copilot 代理人模式實戰」

## 總結

多奇教育訓練網站提供了完整的 GitHub Copilot 課程體系，從基礎入門到進階應用，從 IDE 整合到終端機使用，以及最新的 AI Agent 模式，涵蓋了 GitHub Copilot 的各個面向。

總計 4 門課程，共 7 部影片，總時長超過 20 小時的豐富內容，適合不同程度的開發者學習。
